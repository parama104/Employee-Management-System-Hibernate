# Employee Management System using Hibernate (JPA)

## Project Description

This is a Java console-based Employee Management System developed using Hibernate (JPA) and MySQL. It performs CRUD (Create, Read, Update, Delete) operations on employee records.

## Technologies Used

- Java
- Hibernate (JPA)
- Maven
- MySQL
- Eclipse IDE

## Features

- Add Employee
- View All Employees
- Search Employee by ID
- Update Employee Details
- Delete Employee
- Exception Handling
- Salary Validation
- Duplicate Employee ID Check
- Menu-Driven Console Application

## Database

Database Name:
employee_db

Table:
employee

## Concepts Covered

- Entity
- EntityManager
- EntityManagerFactory
- EntityTransaction
- JPQL
- CRUD Operations
- Maven
- Hibernate Configuration
- Exception Handling
